# FastAPI Contacts CRUD API

A simple REST API for managing contacts using **FastAPI** and **SQLite**.

## Features
- Create a new contact
- View all contacts
- Get a contact by ID
- Update contact details
- Delete a contact

## Tech Stack
- FastAPI
- SQLite
- Uvicorn
